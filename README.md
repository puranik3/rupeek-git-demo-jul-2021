# Git and GitHub
How to use Git and GitHub - A short introduction

## Steps to initialize repo
$> cd repo
$> git init
$> git status

## Steps to commit
```
git add <file>
git restore --staged <file>
git commit -m "commit message"
git log
```

## Steps to branch
```
git branch <branchname>
git branch
git checkout <branchname>
```
